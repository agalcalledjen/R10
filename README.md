## Installation

NOTE: NodeJS is required and yarn is recommended.

Clone this project.

```
git clone < project directory .git >
```

### Setup

#### iOS

Install the dependencies.

```
yarn install
react-native run-ios
```

or

```
npm install
```

Run the project.

```
react-native run-ios
```

---

#### Android

Install the dependencies.

```
yarn install
```

or

```
npm install
```

Start the emulator.

Run the project.

```bash
react-native run-android
```
