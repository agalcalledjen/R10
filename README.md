## Installation

**NOTE: Yarn is recommended.**

Clone this project.

```
git clone < project directory .git >
```

### Setup

#### iOS

Install the dependencies.

```
yarn install
```

Run the project.

```
react-native run-ios
```

---

#### Android

Install the dependencies.

```
yarn install
```

Start the emulator (Android 9 (Pie) SDK is recommended).

Run the project.

```bash
react-native run-android
```
