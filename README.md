## Installation

NOTE: NodeJS is required and yarn is recommended.

1. Clone this project.

```bash
git clone < project directory .git >
```

2. Setup

### iOS

1. Install the dependencies.

```bash
yarn install
react-native run-ios
```

or

```bash
npm install
```

2. Run the project.

```bash
react-native run-ios
```

---

### Android

1. Install the dependencies.

```bash
yarn install
```

or

```bash
npm install
```

2. Start the emulator.

3. Run the project.

```bash
react-native run-android
```
